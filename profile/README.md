# Wagtail

Wagtail is an open source content management system built on Django, with a strong community and commercial support. It's focused on user experience, and offers precise control for designers and developers.

:sparkle: View the [official Wagtail public roadmap](https://wagtail.org/roadmap/).

## Learn more

* [wagtail.org](https://wagtail.org/) is our main website.
  * We [blog about our work](https://wagtail.org/blog/) there.
  * We [publish a weeklyish newsletter](https://wagtail.org/newsletter/).
* [docs.wagtail.org](https://docs.wagtail.org/en/stable/) for developers documentation.
* [guide.wagtail.org](https://guide.wagtail.org/) for CMS users documentation.
